# 🍔 FoodExpress

FoodExpress is a modern food delivery platform that allows users to browse restaurants, explore menus, place orders, and track deliveries seamlessly. The project aims to provide a fast, user-friendly, and efficient food ordering experience.

## 🚀 Features

### User Features
- User Registration & Login
- Browse Restaurants
- Search Food Items
- View Restaurant Menus
- Add Items to Cart
- Place Orders
- Order Tracking
- Order History
- Responsive Design

### Admin Features
- Manage Restaurants
- Manage Food Items
- View and Manage Orders
- Update Order Status
- Dashboard Analytics

## 🛠️ Tech Stack

### Frontend
- Next.js
- React.js
- Tailwind CSS
- TypeScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT (JSON Web Tokens)
